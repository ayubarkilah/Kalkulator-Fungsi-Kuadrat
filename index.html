Quadratic function (Fungsi kuadrat)

fungsi kuadrat adalah sebuah fungsi polinomial atau suku banyak

dimana pangkat tertinggi dari variable atau perubahnya adalah 2

Bentuk umum persamaan kuadrat adalah:

ax^2 + bx + c = 0

Sedangkan bentuk umum dari fungsi kuadrat adalah:

f(x) = ax^2 + bx + c

Dengan a, b, merupakan koefisien, dan c adalah konstanta.
